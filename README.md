# Задание 1

## Задача 1

### Текст задачи

Решить задачу из лабораторной работы №2 (Задание 1. List.map) для
последовательности

### Алгоритм решения

Пользователь вводит числа через пробел. Далее строка разбивается на отдельные элементы с помощью встроенной функции Split, после чего получается массив строк.
Затем используется функция Seq.choose, которая проверяет каждый элемент и пытается преобразовать его в число с помощью функции Int32.TryParse. Если преобразование прошло успешно, число добавляется в последовательность, иначе элемент пропускается.
После этого используется функция Seq.map, которая применяется к каждому элементу последовательности. Для каждого числа выполняется проверка на чётность с помощью операции нахождения остатка от деления на 2.
Если остаток равен нулю, возвращается значение true, иначе — false.

## Задача 2

### Текст задачи

Решить задачу из лабораторной работы №2 (Задание 2. List.fold) для
последовательности

### Алгоритм решения

Пользователь вводит римские числа через пробел. Далее строка разбивается на отдельные элементы с помощью функции Split, после чего формируется последовательность строк.
Для проверки корректности ввода используется функция Seq.choose. Для каждого элемента вызывается функция romanToInt, которая преобразует римское число в соответствующее десятичное значение. Если введённая строка не соответствует римским числам от I до IX, она пропускается.
После этого используется функция Seq.fold, которая выполняет последовательное суммирование элементов.
Функция Seq.fold принимает начальное значение, равное 0, и функцию addRoman. Внутри функции addRoman текущее значение суммы прибавляется к числу, полученному из функции romanToInt.

## Задача 3

### Текст задачи

Вывести количество файлов, имя которых начинается с заданного
символа, в указанном каталоге и его подкаталогах.

### Алгоритм решения

Пользователь вводит путь к каталогу. Далее выполняется проверка существования указанной папки с помощью функции Directory.Exists.
Если папка существует, пользователь вводит символ, с которого должно начинаться имя файла.
После этого с помощью функции Directory.GetFiles получается список всех файлов в указанном каталоге и его подкаталогах. Для поиска во всех вложенных папках используется параметр SearchOption.AllDirectories.
Полученный список файлов преобразуется в последовательность, после чего используется функция Seq.filter. Она выбирает только те файлы, имя которых начинается с заданного символа. Для получения имени файла используется функция Path.GetFileName.
Затем с помощью функции Seq.length подсчитывается количество найденных файлов.
